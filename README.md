## 前言
### 学习目的
  本书的目的是要解释所有计算机系统的本质概念，并展示这些概念是如何影响应用程序的正确性、性能和实用性的。是为了使我们写的程序能够更好的利用操作系统和软件系统提供的功能，对各种操作条件和运行时参数都能正确操作，运行起来更快，并能避免出现使程序容易受到网络攻击的缺陷。
### 背景知识
  本书的重点是执行x86-64(即基于x86指令集架构的64位处理器)机器代码的系统。
  并且本书考虑的是在Linux操作系统上运行C语言程序。
